## 🚀 快速开始

### 方式一：直接打开（最简单）

```
双击根目录的 index.html 文件即可在浏览器打开，
然后选择你想体验的版本！
```

### 方式二：使用HTTP服务器（推荐）

#### 方法A：使用 Node.js
```bash
cd 交互式演示平台
npm install
npm start
# 访问 http://localhost:8080
```

#### 方法B：使用 Python
```bash
cd 交互式演示平台
python -m http.server 8080
# 访问 http://localhost:8080
```

#### 方法C：一键启动（Windows）
```
双击根目录的 一键启动.bat 文件
```

## 🚀 部署到 GitHub Pages

### 自动部署（推荐）

项目已配置 GitHub Actions，只需推送到 `main` 分支即可自动部署！

### 步骤

1. 在 GitHub 创建仓库
2. 推送到 GitHub
```bash
cd 交互式演示平台
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```
3. 启用 GitHub Pages：
   - 仓库 Settings → Pages
   - Source：`main branch` → `/root`
   - 保存后等待 1-5 分钟

### 访问地址

```
https://你的用户名.github.io/你的仓库名/
```
